# homelab-portfolio

This repository documents the design, deployment, administration, and
security monitoring of my personal homelab.

## Objectives

- Build experience with Linux and Windows administration
- Practice virtualization and containerization
- Configure segmented networks and firewall rules
- Deploy security monitoring and logging tools
- Practice incident detection and response
- Automate repetitive administrative tasks

## Lab Environment

- Ubuntu Server virtual machines
- Windows 11 virtual machine
